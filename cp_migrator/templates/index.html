How to Use
  1. The tool auto-scans the TAP directory on load.
  2. Select the Virtual Systems to migrate.
  3. Optionally adjust interface name mappings (CP → PAN).
  4. For wave migration: upload an existing Panorama XML and enable append mode.
  5. Click Convert — review the report and download the XML.
  6. Import the XML into Panorama via Device → Setup → Operations → Import named configuration snapshot.
